# 3D Photo Synthesis Engine

> 從一對 **彩色圖 + 深度圖**，合成可在瀏覽器中即時拖曳旋轉的 **3D 照片**（類似 Facebook 3D Photo）。

一切運算在你自己的機器完成——無需雲端、無需帳號。後端計算 3D 網格，瀏覽器用 WebGL 即時渲染與互動。

---

## ✨ 特色

- 🖼️ **輸入單純**：一張 RGB 圖 + 一張深度圖即可生成 3D 照片
- 🖱️ **即時互動**：滑鼠拖曳旋轉視角、滾輪縮放，順暢無延遲（渲染在瀏覽器 WebGL）
- 🎚️ **可調效果**：斷邊靈敏度與視差強度可調，控制立體感與邊緣裂開程度
- 🔒 **本機運算**：影像不離開你的機器，無上傳雲端
- 🌐 **跨平台**：純網頁前端，可在任何現代瀏覽器使用（桌機 / 筆電）

## 🚀 開始使用

### 需求

- **Windows**（提供一鍵腳本）或任何能跑 Python 的系統
- [Python](https://www.python.org/downloads/) 3.10 以上
- [Node.js](https://nodejs.org/) 18 以上
- 一張現代瀏覽器支援 WebGL（Chrome / Edge / Firefox 皆可）

### 安裝與啟動（Windows 一鍵）

1. 下載或 clone 本專案。
2. 雙擊 **`engine.bat`**，選擇 `1. install`（首次安裝，會自動建立環境並安裝依賴）。
3. 安裝完成後，再次執行 `engine.bat`，選擇 `3. run`。
4. 瀏覽器開啟 **http://127.0.0.1:5173**。

> 命令列也可：`engine.bat install` 然後 `engine.bat run`。

### 其他系統（手動）

```bash
# 後端
python -m venv .venv
.venv/bin/pip install -r requirements.txt          # Windows: .venv\Scripts\pip
.venv/bin/python -m uvicorn backend.app:app        # 後端啟動於 :8000

# 前端（另開一個終端機）
cd frontend && npm install && npm run dev          # 前端啟動於 :5173
```

## 📖 操作流程

1. 開啟頁面後，分別選擇 **RGB 圖片** 與 **深度圖**。
2. （選用）展開「進階參數」，調整：
   - **斷邊百分位**：值越高，越少邊緣被裂開（畫面越完整、立體斷層越少）。
   - **視差強度**：值越大，前後景的立體分離越明顯。
3. 按 **「合成 3D 照片」**，稍候片刻即會出現 3D 模型。
4. **滑鼠左鍵拖曳**旋轉、**滾輪**縮放、**右鍵拖曳**平移。

### 深度圖怎麼來？

深度圖是一張與彩色圖同視角、以灰階表示「遠近」的圖片（越亮通常代表越近或越遠，依來源而定）。
可由 AI 深度估算工具（如 MiDaS、Depth Anything）從單張照片生成，或由具備深度感測的相機取得。

## ❓ 常見問題

**Q：合成後畫面出現尖刺或破洞？**
A：嘗試調高「斷邊百分位」減少裂開，或確認深度圖與彩色圖是否對齊、同一視角。

**Q：支援哪些圖片格式？**
A：RGB 支援 PNG / JPG；深度圖支援 PNG（8 或 16 位元）與 TIFF。RGB 與深度圖解析度不同時會自動對齊。

**Q：我的影像會被上傳到網路嗎？**
A：不會。前後端都跑在你本機，影像只在本機處理。
